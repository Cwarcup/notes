# Rest Conventions

| Methods	|       URI     |      result    |
| ------- | ------------- | -------------- |
|  GET   	|  /posts       | retrieve all posts |
| ------- | ------------- | -------------- |
|  GET  	|  /posts/:id   | retrieve posts with the given ID |
| ------- | ------------- | -------------- |
|  POST   |  /posts       |  create a new post |
| ------- | ------------- | -------------- |
|  PUT    |  /posts/:id   |  update a post |
| ------- | ------------- | -------------- |
|  DELETE |  /posts:id    | deletes a post |
